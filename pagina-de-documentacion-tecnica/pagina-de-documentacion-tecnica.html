CSS

Introducción al CSS

CSS (Cascading Style Sheets) es el lenguaje que usamos para darle estilo a las páginas web. Si el HTML es el esqueleto, el CSS es la ropa, el maquillaje y toda la onda. Sirve para cambiar colores, tamaños, posiciones, fuentes y mucho más.

Ejemplo básico:

  • HTML:
    • <p class="importante">¡Hola mundo!</p>

  • CSS:
    • .importante { color: red; font-size: 24px; }

    Selectores CSS

    Los selectores son la forma en que CSS elige a qué elementos aplicar los estilos. Podés seleccionar por tipo, clase, id, atributos y más.

    Ejemplos:

  • Elemento:
    • p { color: blue; }

  • Clase:
    • .titulo { font-weight: bold; }

  • ID:
    • #principal { background-color: yellow; }

  • Selector descendente:
    • nav a { text-decoration: none; }

    Colores y unidades

    CSS te deja jugar con muchos tipos de colores y unidades para tamaño y espaciado. Hay colores con nombre, hexadecimales, RGB y HSL. Y las unidades pueden ser fijas o adaptables.

    Ejemplos:

    h1 {
    | color: #ff5733; /* naranja */
    | font-size: 2em; /* 2 veces el tamaño del elemento padre */
    | margin: 20px; /* espacio con píxeles */
    }

    Modelo de Caja (Box Model)

    Todo en CSS es como una cajita: contenido, padding (relleno interno), border (borde) y margin (espacio hacia afuera). Con box-sizing controlás si el tamaño incluye o no esos espacios.

    Ejemplo:

    div {
    | width: 200px;
    | padding: 20px;
    | border: 5px solid black;
    | margin: 10px;
    | box-sizing: border-box; /* incluye padding y border en el width */
    }

    Textos y Tipografías

    Podés cambiar la fuente, el tamaño, el grosor, el interlineado, etc. Es clave para que tu página no se vea como si saliera de 1998.

    Ejemplo:

    body {
    | font-family: 'Arial', sans-serif;
    | font-size: 16px;
    | line-height: 1.5;
    | text-align: center;
    }

    Posicionamiento

    Controla dónde aparece un elemento. El position cambia cómo se comporta en relación a otros elementos.

    Ejemplo:

    .caja {
    | position: absolute;
    | top: 50px;
    | left: 100px;
    }

    Display y Layouts

    display define cómo se muestran los elementos. block, inline, flex y grid son los más comunes. flex y grid te dan poderes mágicos para hacer layouts responsive.

    Ejemplo con Flexbox:

    .contenedor {
    | display: flex;
    | justify-content: space-between;
    | align-items: center;
    }

    Responsive Design (Diseño adaptable)

    Diseñá para que tu web se vea bien en todos los dispositivos. Con @media podés aplicar estilos según el tamaño de la pantalla.

    Ejemplo:

    @media (max-width: 600px) {
    | body { background-color: lightblue; }
    }

    Animaciones y Transiciones

    Dale vida a tu sitio con transiciones suaves y animaciones personalizadas. Podés cambiar colores, posiciones, opacidades, ¡lo que quieras!

    Ejemplo:

    .boton {
    | transition: background-color 0.3s;
    }

    .boton:hover {
    | background-color: green;
    }

    Buenas prácticas y organización

    Escribí CSS limpio, ordenado y fácil de mantener. Usá clases reutilizables, nombres claros, y si podés, metete con metodologías como BEM. Separá tu CSS en archivos y mantené tu código DRY (Don't Repeat Yourself).

    Ejemplo de clase reutilizable:

    .btn {
    | padding: 10px 20px;
    | background: black;
    | color: white;
    | border: none;
    }
